## Description
A command-line calculator built in Python that performs basic 
arithmetic operations with proper input validation and error handling.

## Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- Handles invalid (non-numeric) inputs gracefully
- Handles division by zero with a clear error message
- Menu-driven loop — keeps running until user chooses to exit

## How to Run
Make sure Python is installed, then run:
```bash
python calculator.py
```
